    【簡介】
    我是徐琮偉，也可以叫我 Eason；居住台北市內湖區，平時興趣唱歌、踏青看風景、陪家人逛夜市；曾因熱愛跑步跑到受傷、而改以飛輪運動居多。

    如同我的興趣，喜歡發掘新奇事物、挑戰極限，視覺設計出身卻不安於舒適、前端之路難走卻闖入，只為追求製作潛能無限的網頁，追求挑戰成功後的豐碩果實，使自己的貢獻促進公司成功與茁壯。

    從事前端程式 6 年、網頁設計 11 年，經歷卜過的行業有旅行、物流、幼教、3C、電影社群，及線上英語學習，累積的經歷將讓我在 貴公司更快速熟悉公司商業模式，更快速於職務上貢獻我所能！


    【網頁前端程式 6 年經歷】
    職涯中倚自己對網頁技術發展的認識與實作，認知前端技術對網頁開發各階段而言「價值貢獻」的舉足輕重，能成為主要功能的開發與維繫的重要腳色，使我對前端職務充滿熱忱、讓我有不斷吸收與更新開發技術的原動力！


    【使用 Vue cdn & cli 開發網站，修改 React 現有專案能力】
    前端是個技術日益更迭的職業，開發框架出現與盛行許久，跟上主流技術會是我需要配賦的基本能耐；目前公司開發環境讓我能以Vue cdn 方式進行開發、也已擁有實作上限的作品，Vue cli 開發則有自己平日練習的一個範例網站；React 也有上過學程，於現職上擁有維護現有專案能力。


    【學習 PHP & Node + firebase 後端語言增進團隊溝通】
    工作經驗體認溝通的重要，溝通的順暢也代表做事效率，也建構愉悅的辦公氣氛。為了能與後端人員溝通無阻，也粗淺有涉略屬後端領域的 php 知識，以及簡易 Node.js + Firebase 環境的部落格。

    【於 50 人會議主講網頁功新能介紹、開課分享新學技術】
    一間企業的成功不會是「一個」技術封頂的人可維繫，讓更多的人成長發光才會是成功能長久之計；
    程式人員與電腦對話一整天是日常，最初只想藉演講與教學鍛鍊社交，而爭取了在人群前漏臉的機會－你可以在會議上直接取得使用者們的寶貴意見，你可以在會後看到夥伴使用了新技術，進而讓開發變得更有效率，收穫超乎意料且充滿魅力！


    【網頁設計出身，擁有扎實的介面設計經驗】
    求學階段學的是視覺設計，一直對網頁領域有所涉略，而以「網頁設計」職務進入社會，從而體認前端職務在開發階段中的舉足輕重價值，深深被吸引著，也因此提供我學習熱誠，使我在職場有機會引用前端技術於專案，於多年前的職務中由「網頁設計」轉任「網頁前端工程師」，無縫接軌。


    【擁有一定層度的個人魅力，凝聚人群】
    所謂近朱者赤，職場上總有正派且健談的人物、充滿魅力，從認識進而學習與人相處進退的經驗總是珍貴，不知不覺也發現自己有凝聚人群的能耐，我想我是能讓工作獻上愉悅氣氛的人。
